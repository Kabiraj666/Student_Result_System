Interactive CLI Emulator

This terminal simulates the compiled Java executable running on a server. It interacts directly with the same in-memory database as the dashboard above!

💡 Test These Commands:

  • Type 2 to see the formatted list of students in the console.
  • Type 1 to add a new student step-by-step.
  • Type 5 to record course scores (e.g. CS201, Data Structures, 4 credits, 85 marks) and check letter grade computation.
  • Type 9 to view mathematical statistics calculated directly in Java.

💻 Backend Architecture

When you enter values, they are run through Java-equivalent OOP validation pipelines. Try typing invalid marks (e.g., 150) or negative credits (-2) to watch custom exceptions get caught and printed!
